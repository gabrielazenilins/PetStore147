Testes de API / Serviços / Integração de Sistemas
 
O que é uma API?
Interface de comunicação e processamento entre aplicações/Sistemas
 
Tipos de API:
- Web Services: XML --> Correios (Consultar Preço e Prazo)
- REST/RESTfull: JSON --> IFood, Uber
 
Tipos de Documentação:
- Nenhuma
- Manual
- Manual convertida/traduzida
- Documentação automática (Swagger)
 
Como faremos aqui no curso?

Ok - Realizar testes via Postman (quase dizer que é por configuração)
